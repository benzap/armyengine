\subsection{Detailed Description}
Wrapper to store varying functions as one function type. 

Wrap function\-Templates into function\-Event\-Templates Wrap function\-Base\-Templates into function\-Event\-Templates

Used to wrap the other given function types into the function\-Event\-Template type that is stored within the callback manager.

The wrapper converts zero-\/paramter and one-\/parameter functions into the two-\/paramter function used within the callback manager, and assigns the unused parameters as zero-\/values.
