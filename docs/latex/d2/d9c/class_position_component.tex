\subsection{Detailed Description}
Used to describe the position of an entity. 

The position of the entity is important for the \hyperlink{class_sprite_component}{Sprite\-Component}, \hyperlink{class_text_component}{Text\-Component}, \hyperlink{class_shape_component}{Shape\-Component} and Collision Component.
