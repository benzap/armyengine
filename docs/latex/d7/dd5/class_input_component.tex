\subsection{Detailed Description}
Used to give an entity control over specific inputs from the user. 

Input component is a special \hyperlink{class_state_component}{State\-Component} that will trigger a callback when a provided input is used.

Provided inputs include \char`\"{}\-Keyboard\char`\"{} and \char`\"{}\-Mouse\char`\"{} Depending on which one is chosen, the resulting attributes searched will also vary.
